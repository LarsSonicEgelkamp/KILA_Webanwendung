# KILA Webanwendung

Diese Anwendung ist eine einfache React-TypeScript-Anwendung.

## Entwicklung

1. Abhängigkeiten installieren (erfordert Internetzugang):
   ```bash
   npm install
   ```
2. Entwicklungsserver starten:
   ```bash
   npm run dev
   ```
3. Tests ausführen:
   ```bash
   npm test
   ```

Die Startseite (Home) wird aus der `main`-Datei geladen und über die linke Navigationsleiste können weitere Seiten erreicht werden.
